## Required Dependencies

```bash
npm install dotenv        # To load environment variables
npm install axios         # HTTP client for making API requests
npm install html-literal  # Template literal tag for HTML
npm install lodash        # Utility library (use camelCase function)
npm install navigo        # Client-side router for SPA navigation
npm install parcel        # Bundler that builds and serves your app
```

Or install all at once:

```bash
npm install dotenv axios html-literal lodash navigo parcel
```
